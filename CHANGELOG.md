# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-02-16

### Changed

- **Major refactoring**: Restructured codebase into modular architecture
- Split monolithic script into focused modules (core, cli, utils)
- Improved code organization and maintainability

### Added

- Complete test suite with pytest
- Automated build system with GitHub Actions
- Package configuration (setup.py, pyproject.toml)
- Development dependencies management
- Code quality tools (Black, Flake8, MyPy)
- Comprehensive documentation (CONTRIBUTING.md)
- Build automation script
- CI/CD pipelines for testing and releases

### Improved

- Better error handling and logging
- Enhanced CLI user experience
- More flexible configuration options
- Cleaner API for programmatic use

## [1.0.0] - Previous

### Features

- Bottom-right watermark removal
- Smart edge text detection
- Web UI interface
- Desktop executable
- Batch processing support
- Recursive folder processing
